# EQE: Evaluating the Quality of Explanations
> **Project P4** — *Explainable and Trustworthy AI* Course

This repository contains the work for the "Evaluating the Quality of Explanations" project. The goal is to investigate existing XAI methods, implement a framework for testing, and propose hybrid explanation approaches.

---

##  Roadmap

### Step 1: Research & Data
- [ ] **Data Selection:** Identify and acquire suitable datasets.
- [ ] **State of the Art:** Review existing methods for model explanation.

### Step 2: Implementation
- [ ] **Framework Development:** Build a testing environment to evaluate XAI methods.

### Step 3: Innovation
- [ ] **Hybrid Methods:** Research and develop novel explanation techniques (e.g., combining multiple existing methods).

### Step 4: Documentation
- [ ] **Finalizing:** Prepare the supporting document and the final presentation.
